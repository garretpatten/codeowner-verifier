# Code of Conduct

We want this project to stay small, friendly, and useful. A few simple rules:

## Expected behavior

- Be respectful and constructive, especially in issues and pull requests.
- Assume good intent; disagree with ideas, not people.
- Keep feedback specific (repro steps, logs, CODEOWNERS snippets) so others can help.

## Unacceptable behavior

Harassment, discrimination, threats, sexualized language or attention, sustained disruption, doxxing, or other behavior that would be out of place in a professional setting is not allowed.

## Reporting

If something crosses that line, email [garret.patten@proton.me](mailto:garret.patten@proton.me). Reports are read in confidence; the maintainer will respond as promptly as they can.

## Enforcement

The maintainer may remove or moderate content (for example issue comments) and block repeat offenders from interacting with this repository if needed.
